# ROS机器人编程学习实践

### 功能包中分别有README运行教程文件

* src - 存放功能包源码
* devel - 开发过程中生成的头文件、可执行文件等...
* build - 编译过程中生成的文件
* install - 开发过程中需要下载的文件
* books - 开发过程记录文档
